Introduction

HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web. Il permet de structurer le contenu et de définir la mise en forme de la page.

Les balises HTML sont des éléments de base qui décrivent la structure et le contenu des pages web. Chaque balise a une signification spécifique et un rôle particulier.

Balises de Base

Les balises de base sont fondamentales en HTML pour structurer le contenu :

  • <h1> à <h6> : Balises de titre, de <h1> (le plus important) à <h6> (le moins important).
  • <p> : Balise de paragraphe pour le texte courant.
  • <br> : Balise de saut de ligne, utilisée pour insérer un saut de ligne sans paragraphe.
  • <strong> : Balise pour le texte important ou en gras.
  • <em> : Balise pour le texte mis en emphase ou en italique.
  • <div> : Balise de division utilisée pour grouper des éléments pour le style ou le script.
  • <span> : Balise de texte en ligne utilisée pour le style ou les scripts sans ajouter de structure.

Exemples :

Exemples d'utilisation des balises :


      <h1>Titre Principal</h1>
      <p>Ceci est un paragraphe de texte.</p>
      <br>
      Ce texte sera sur une nouvelle ligne après le saut de ligne.
      <strong>Texte en gras</strong>
      <em>Texte en italique</em>
      <div>Contenu divisé pour le style.</div>
      <span style="color: red;">Texte en rouge</span>

Liens

Les liens permettent de naviguer entre différentes pages web ou de rediriger vers d'autres sites. Voici comment créer un lien :

Cliquez ici pour visiter Example.com

Exemple de code :

<a href="https://www.example.com">Cliquez ici pour visiter Example.com</a>

Exemple de code :

Listes

Il existe deux types de listes en HTML : les listes ordonnées et les listes non ordonnées.

Liste Ordonnée

  1. Élément 1
  2. Élément 2
  3. Élément 3

Liste Non Ordonnée

  • Élément A
  • Élément B
  • Élément C

Exemple de code :


<ol>
  <li>Élément 1</li>
  <li>Élément 2</li>
  <li>Élément 3</li>
</ol>

<ul>
  <li>Élément A</li>
  <li>Élément B</li>
  <li>Élément C</li>
</ul>

Tableaux

Les tableaux permettent de structurer les données en lignes et en colonnes. Ils sont souvent utilisés pour afficher des données tabulaires de manière organisée.

Un tableau HTML se compose des éléments suivants :

  • <table> : Élément conteneur principal pour le tableau.
  • <thead> : Contient les en-têtes de colonnes.
  • <tbody> : Contient le corps du tableau avec les données.
  • <tr> : Représente une ligne dans le tableau.
  • <th> : Définit une cellule d'en-tête dans le tableau.
  • <td> : Définit une cellule de données dans le tableau.

Voici un exemple de tableau simple :

Liste des étudiants inscrits
Nom Âge Ville
Jean 30 Paris
Marie 25 Lyon

Exemple de code :


      <table>
        <caption>Liste des étudiants inscrits</caption>
        <thead>
          <tr>
            <th>Nom</th>
            <th>Âge</th>
            <th>Ville</th>
          </tr>
        </thead>
        <tbody>
          <tr>
            <td>Jean</td>
            <td>30</td>
            <td>Paris</td>
          </tr>
          <tr>
            <td>Marie</td>
            <td>25</td>
            <td>Lyon</td>
          </tr>
        </tbody>
      </table>

Exemple de code :


      <form>
        <div>
          <label for="name">Nom :</label>

Éléments Sémantiques

Les éléments sémantiques donnent un sens clair et spécifique au contenu de la page. Voici quelques exemples :

  • <header> : Entête de page ou de section.
  • <nav> : Navigation.
  • <main> : Contenu principal de la page.
  • <footer> : Pied de page.
  • <article> : Contenu autonome pouvant être distribué indépendamment.
  • <section> : Section thématique de la page.
  • <aside> : Contenu secondaire ou complémentaire.
